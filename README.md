# LeanFEM

**LeanFEM** — это MATLAB-реализация метода конечных элементов для решения задач теории упругости, использующая моментные элементы.

## Возможности
### Решение различных типов задач
- Объёмные задачи с трёхмерным напряжённым состоянием
- Плоские задачи - плоская деформация, плоское напряжение
- Стационарные и нестационарные задачи
- Явные и неявные динамические решатели

### Гибкая архитектура
- Модульная система для простого добавления новых физических формулировок, материалов, типов конечных элементов и форматов файлов сеток
- Поддержка различных типов элементов — четырёхугольные элементы (2D), гексаэдральные элементы (3D) с полным и сокращённым интегрированием

### Визуализация и анализ результатов
- Интегрированное отображение сетки, граничных условий и нагрузок
- Визуализация полей перемещений, деформаций и напряжений

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/LeanFEM.git
cd LeanFEM
```
2. Добавьте проект в путь поиска: запустите src/setup.m
